Frontend Créditos

Interface moderna desenvolvida em Angular 20 para consumir a API de consulta de créditos

📁 Ver no GitHub ⬇️ Download ZIP

Sobre o Projeto

O Frontend Créditos é uma aplicação web moderna desenvolvida com Angular 20, projetada para oferecer uma interface intuitiva e responsiva para consulta de créditos através da API.

Com design responsivo e experiência de usuário otimizada, o projeto demonstra as melhores práticas de desenvolvimento frontend, incluindo testes automatizados e containerização com Docker.

Como Usar

1

Clone o Repositório

git clone https://github.com/swilliamsilva/frontend-creditos.git

Ou faça o download do ZIP e extraia os arquivos

2

Instale as Dependências

cd frontend-creditos
npm install

Instala todas as dependências do projeto

3

Execute o Servidor

ng serve

Acesse em http://localhost:4200

4

Execute os Testes

ng test

Executa testes com Karma e Jasmine

Deploy com Docker

🐳 Build da Imagem

docker build -t frontend-creditos .

🚀 Executar Container

docker run -p 80:80 frontend-creditos

Após executar os comandos acima, acesse a aplicação em http://localhost
